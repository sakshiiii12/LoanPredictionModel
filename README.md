# 🏦 Loan Default Prediction App

A Machine Learning web application built using **Streamlit** that predicts whether a loan will be fully paid or not based on applicant financial details.

---

## 🚀 Project Overview

This project uses a classification model trained on a loan dataset to predict:

- ✅ Loan Fully Paid (Low Risk)
- ⚠ Loan Not Fully Paid (High Risk)

The application allows users to enter applicant details and get instant predictions.

---

## 📊 Dataset Features

The model was trained using the following features:

- credit.policy  
- purpose  
- int.rate  
- installment  
- log.annual.inc  
- dti  
- fico  
- days.with.cr.line  
- revol.bal  
- revol.util  
- inq.last.6mths  
- delinq.2yrs  
- pub.rec  

Target Variable:
- not.fully.paid (0 = Fully Paid, 1 = Default)
